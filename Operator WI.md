# Operator WI（玩家操作手冊）

## 遊戲目標

將單字魚拖入正確語法水缸：

- **Noun** 名詞  
- **Verb** 動詞  
- **Adjective** 形容詞  
- **Adverb** 副詞  

---

## 遊戲模式總覽

在主畫面選擇模式後按 **Start Game**。

| 模式 | 說明 | 適合對象 |
|---|---|---|
| **Zen 禪修** | 依設定抽取單字（預設 30 題），無限時，無計分。 | 初學者、慢慢練 |
| **Time Attack 限時** | 計分 + 連擊，畫面常態 10 條魚，有回合倒數。 | 想挑戰手速與反應 |
| **Static Drill 靜態練習** | 每次一隻魚，共 30 題，自適應加權出題。 | 針對弱點集中練習 |
| **Sandbox 沙盒** | 每次一隻靜止魚，有漸進提示，不記錄成績。 | 建立字尾肌肉記憶 |

---

## 快速開始

1. 在主畫面 **遊戲模式** 選擇模式
2. 按 **Start Game**
3. 按住魚拖放到上方正確的 Tank
4. 視模式決定結束條件（見各模式說明）

---

## 各模式詳細說明

### Zen 禪修模式

- 預設 30 題（可在 Settings 調整每局單字數）。
- 無時限，無分數，拖錯立即顯示正確位置後移除。
- 全部完成後進入結算，顯示準確率與弱點字尾分析。
- 拖錯的單字會在背後**調高出現權重**，下次練習時更常出現。

### Time Attack 限時模式

- 畫面同時維持 **10 條魚**，拖走一隻即補一隻。
- 難度決定時間上限：Easy 180s／Normal 120s／Hard 90s。
- **計分規則**
  - 答對：+10 分；連擊加成每層 +2（第 2 連擊 +12、第 3 連擊 +14……）
  - 答錯：−5 分
- **連擊（Combo）**：每次答對重置 3 秒連擊窗；逾時或答錯則歸零。
- 時間到或手動退出後進入結算。

### Static Drill 靜態練習模式

- 每次畫面只出現 **1 條魚**，答對或答錯後換下一條，共 30 題。
- **自適應加權出題**：
  - 答錯次數多 → 出現機率上升（答錯一次 +5 權重）
  - 答對次數多 → 出現機率下降（答對一次 −2 權重）
  - 已熟練（連對 3 次）→ 權重降至 1（幾乎不再重複）
- 結算頁顯示**掌握度進度**（每個詞彙的熟練狀況）。

### Sandbox 沙盒模式

- 每次一條魚，靜止在畫面中央，無題數上限，隨時可退出。
- **漸進提示系統**（每條魚重置計時）：

  | 時間 | 提示狀態 |
  |---|---|
  | 0–2 秒 | 無提示，考驗瞬間反應 |
  | 2 秒後 | 各 Tank 顯示對應字尾（如 `-ness`, `-tion`） |
  | 5 秒後 | 正確 Tank 發出黃色光暈 |

- **不計入**學習歷程或單字統計；所有動作記錄於 **Sandbox Log** 供另行分析。
- 答對或答錯後，提示清除，立刻出現下一條魚。

## 操作方式

| 動作 | 說明 |
|---|---|
| **拖放魚** | 滑鼠拖曳 / 手指按住拖曳 |
| **暫停** | HUD **Pause** 或鍵盤 **ESC** |
| **繼續** | 暫停畫面 **Resume**，或再按 **ESC** |
| **重開** | 暫停或結算畫面 **Restart / Play Again** |
| **退出主畫面** | HUD **退出主畫面** → 確認後回主選單（本局不寫入戰績） |

> **注意**：若教學或浮層視窗開著，**ESC** 會優先關閉該視窗，而非暫停遊戲。

## 主選單功能

| 按鈕 | 說明 |
|---|---|
| **Tutorial 📖** | 字尾速查表；遊戲中開啟**自動暫停**，關閉後恢復 |
| **單字庫 📚** | 新增／編輯／刪除詞條；匯入匯出 JSON；合併 words.json |
| **戰績 📊** | 最近 50 局對局紀錄 |
| **History 📜** | 最近 50 筆排序記錄 + 各單字掌握度進度條 |
| **Sandbox Log 📋** | 查看沙盒詳細記錄，可匯出 JSON 或 CSV |
| **Settings** | 音量、全螢幕、難度、每局單字數、魚速、讀音 |

| 欄位 | Zen / Static Drill | Time Attack | Sandbox |
|---|---|---|---|

## Sandbox Log 分析指引

點選主選單 **Sandbox Log 📋** 查看記錄。每筆資料說明：

| 欄位 | 意義 |
|---|---|
| `timeSec` | 從魚出現到放入的秒數 |
| `hintLevel` | 0 = 無提示（<2s）、1 = 字尾可見（2–5s）、3 = 發亮（≥5s） |
| `isConfidentError` | ⚠ = 在 <2s 內拉到錯誤位置（自信但錯誤） |

- **hintLevel 0 答對**：最佳狀況，純粹靠記憶辨識字尾。
- **hintLevel 3 才答對**：詞尾還不熟悉，需多加練習。
- **isConfidentError**：反應快但方向錯，需特別留意該字尾的混淆點。

匯出 CSV 後可用 Excel 或 Python 篩選弱點單字，配合 Static Drill 加強練習。

## 字尾速查（常見規律）

| 詞性 | 常見字尾 |
|---|---|
| **Noun** | `-tion`, `-sion`, `-ness`, `-ment`, `-ity`, `-er`, `-or`, `-ist`, `-ance`, `-ence` |
| **Verb** | `-ize`, `-ify`, `-ate`, `-en` |
| **Adjective** | `-ful`, `-less`, `-ous`, `-ive`, `-able`, `-ible`, `-al`, `-ary`, `-ic` |
| **Adverb** | `-ly`, `-ward`, `-wise` |

完整字尾表請在遊戲中開啟 **Tutorial 📖**。

## 詞庫管理

- 遊戲首次開啟時自動從 `words.json` 載入詞庫到 localStorage。
- 之後以「單字庫」UI 為主，可隨時新增、刪除、匯入匯出。
- 若需更新至最新 `words.json`，請點「單字庫」→ **合併 words.json**。

---

## 資料說明

所有資料存於瀏覽器 **localStorage**，不上傳至伺服器。  
換裝置或清除 Cookie 後資料不會跟著走；出街同一支手機用同一瀏覽器可延續記錄。  
重要的 Sandbox Log 建議定期從「Sandbox Log」介面**匯出備份**。
